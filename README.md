# Fitness Planner API

Constraint-aware fitness planning API for generating weekly workout plans and revising them based on equipment, pain/discomfort, difficulty, and user preference constraints.

## Planned Stack

- FastAPI
- PostgreSQL
- SQLAlchemy 2
- Alembic
- Pydantic Settings
- pytest

## Development Status

Project bootstrap is in progress. The first phase covers repository setup, local PostgreSQL, FastAPI app scaffolding, and test infrastructure.
